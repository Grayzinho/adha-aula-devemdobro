A.D.H.A

  • O que é o A.D.H.A?

    A.D.H.A (Associação Diamantes do Handebol Andaraí), é um projeto onde trazemos os ensinamentos do esporte handebol para crianças e adolescentes.

  • Quais ensinamentos esse projeto passará?

    Nossos professores ensinarão o handebol de formar prática e divertida junto com disciplina, respeito e companheirismo.

  • Onde o projeto é realizado?

    Nosso projeto é realizado na quadra dos Bandeirantes, nos horarios das 17:00 às 20:30 horas.
